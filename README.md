## Usage

* [Qiita Article](http://qiita.com/xkumiyu/items/9dfe51d2bcb3bdb06da3)

### start dev-server

``` sh
$ npm start
```

### build

``` sh
$ npm run build
```

## Supported versions

* node: 5.8.0
* npm: 3.7.3
* react: 0.14.7
* redux: 3.2.1
* babel: 6.5.2
